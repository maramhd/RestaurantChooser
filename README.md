# Restaurant Chooser App

## How to run the code

1. **Clone the repository:**

   ```bash
    https://github.com/maramhd/RestaurantChooser.git

    cd RestaurantChooser
   ```

2. Install dependencies:

```bash
   npm install
```

3. **Install and align Expo dependencies (important step)**

   ```bash
   npx expo install --fix
   ```

   This command ensures that all Expo-related packages are compatible with each other

4. Start the Expo server:
   npx expo start
5. Run on your device:
   Install "Expo Go" app from Google Play or App Store.

Scan the QR code shown in the terminal.
